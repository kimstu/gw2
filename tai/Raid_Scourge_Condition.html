. 灾厄师 . Scourge

症状灾厄师

  DerBaer [LN]
  泰协内容维护小组
  2020.08.06
  未经作者授权，禁止转载

概述

  • 症状灾厄师 是一套相对容易上手的症状输出配置，由于操作简单，破蔑视、单体恢复能力都很强，在作为需要长时间脱离大团的机制位时，倍受玩家青睐。
  • 症状灾厄师 主要依赖 流血 、 折磨 、 燃烧 以及些许 中毒 制造输出。由于这些症状都需要较长时间才能堆叠起高额层数，因此不适合频繁转阶段的战斗 或 速刷阵容。
  • 该套配置对于如何管理及放置 显形沙影 要求比较高，这是本配置为数不多的难点。想要完全驾驭它，你需要理解技能优先级顺序，以及为什么按此顺序施放技能。
  • 恶疾肆虐 这样的AOE输出技能是 症状灾厄师 实战中最具价值的技能，在伤害测试时并未选用。
  • 由于搭配输出型装备和特性，实战中仅能提供有限的团队辅助能力（ 屏障 过于薄弱，症状清理能力贫乏），但在需要远程风筝 BOSS 时，方能凸显其优势（如对抗萨贝莎时，拉火圈），视实战情形选择搭配特性及技能。

特性 与 技能

  • 通用
  • 鲜血
  • 极限输出

特性

  • 寄生蔓延 ：如果输出环境恶劣，请视情形点出，使用该特性会让输出降低 15% 左右。
  • 死亡恐惧 ：如果队伍中只有一名 灾厄师 ，或者 实战中缺乏生命之力，可以点出。
  • 悲伤预告 ：若完全不使用 惩罚 技能，则点出该特性。

技能

治疗技能（可选）：

  • 召唤血魔 （ 死亡盛宴 ）：可以为本体提供优异的持续回血能力，推荐在输出环境较恶劣 或 无法保持近身作战时携带。
  • 沙尘闪耀 ：可以为小队提供 屏障 ，配合特性 酷虐灼热 可以获取更高的症状输出。

通用技能（可选）：

  • 召唤影魔 （ 鬼影降临 ）：可用于恢复生命之力，并能对目标造成一些削弱型症状，在无法经常保持近身作战时，推荐使用此技能。
  • 骸骨仆从 （ 腐烂尸爆 ）：如果团队中有多名 症状灾厄师 ，可以选用帮助团队中的其他 症状灾厄师 恢复生命之力。
  • 痛苦路径 ：可用于堆叠 燃烧 、为自己和周围盟友添加 迅捷 和 稳固 ，同时也可用来解除昏迷。
  • 恶疾肆虐 ：需要更强力的AOE伤害能力 或 与其他唤灵师配合反传染（伤害大砍），可以视情形将 召唤影魔 或 痛苦路径 换成 恶疾肆虐 。
  • 黄沙膨胀 ：需要快速位移时，可选用。
  • 腐蚀毒云 ：需要摧毁投射物时，可选用。
  • 鬼灵缚爪 ：需要聚怪时，可选用。

精英技能（可选）：

  • 血肉魔像 （ 冲锋陷阵 ）：需要提升破蔑视能力 或 无法保持近身作战 时选用。
  • 瘟疫化身 ：症状输出能力更强。

特性说明

牺牲一部分输出能力，以换取 F4将倒地队友拉拽到自己身边，通常用于开荒队伍使用。

特性

  • 疾速渴嗜 ：需要提升移动速度，可以点出。
  • 不洁殉难 ：需要为队友清理症状时，可以点出。

技能

特性

  • 死亡恐惧 ：如果队伍中只有一名 灾厄师 ，或者 实战中缺乏生命之力，可以点出，否则使用 灵魂倒钩 。

配置说明

常规循环说明

症状灾厄师 没有十分固定的技能循环，因为输出时通常只使用一套武器，所有技能的冷却时间都在面板可见，并不必担心输出时卡技能。想要发挥这套配置的最佳输出能力，你需要优先考虑以下问题：

  • 以下技能按优先级顺序冷却即用：
  1. 腐化盛宴
  2. 痛苦浪潮
  3. 死亡束缚
  4. 重压崩溃
  5. 鲜血力量
  6. 显形沙影


  • 节杖的自动攻击链要打满，尤其第三段 腐烂诅咒 ，可以造成很可观的输出。
  • 显形沙影 作为填充技能使用，并尽可能保持三个沙影，以便通过 黄沙贤者 获得 15%的症状持续时间。
  • 施放 沙漠之幕 后获得 传播瘟疫 （该状态会在身上附着一段时间），用来将 鲜血力量 （ 腐化大师 ）给自身的两个症状使用攻击转移出去。尽量避免在施放 鲜血力量 前使用其他腐化技能（如 瘟疫化身 或 恶疾肆虐 ），防止将低伤或无伤的症状转移出去，从而降低输出。 沙漠之幕 → 鲜血力量 技能衔接使用，这两个技能有几乎相同的冷却时间。
  • 如果技能都进入冷却，无法执行上述要求，请使用新一轮的节杖自动攻击链来填充。
  • 如果使用“极限输出”配装， 切换武器 要冷却即用。
  • 战斗中需要着重注意以下两点：

1、放置沙影前，使用惩罚类技能触发 酷虐灼热
2、使用连招 沙漠之幕 → 鲜血力量 触发 传播瘟疫 将 鲜血力量 及 腐化大师 给自身的两个症状使用普攻转移给目标


  • 不要让生命之力累积太多，如果你正在使用特性 死亡恐惧 ，请考虑以下技能优先级：
  1. 沙漠之幕
  2. 炫耀之柱
  3. 邪恶恩惠
  4. 黄沙瀑布
  • 如果你正在使用 灵魂倒钩 ，则考虑以下技能优先级：
  1. 沙漠之幕
  2. 邪恶恩惠
  3. 黄沙瀑布
  4. 炫耀之柱
  • 如果可以持续的保持以上几个要点，你可以在其他技能冷却完毕后立即使用它们。除此之外，实战中需要为了破蔑视保留部分伤害型技能（例如 重压崩溃 ），参照上述说明，以便将输出损失降至最低。

说明

常规：

  • 牢记 德姆火焰 的内置冷却时间为 1秒。
  • 在施放 显形沙影 之前，尽可能使用一个惩罚类技能，用于触发特性 酷虐灼热 。除非你迫切的需要使用 沙尘闪耀 为团队提供 屏障 ，否则不应太随意的施放惩罚技能。本配置可能使用到的惩罚技能包括： 沙尘闪耀 、 痛苦路径 。
  • 如果身上有多余的症状（例如由 瘟疫化身 或 恶疾肆虐 腐化给自己的），可以使用 邪恶恩惠 将它们转化为增益。
  • 瘟疫化身 尽量在 BOSS 不移动时使用。

影魔：

  • 鬼影降临 让你拥有更多生命之力来使用暗影技能，这将变相提升输出。

鲜血特性：

  • 使用 “鲜血” 特性分支将使你具有一定的团队辅助能力，但由于本配置选用输出型装备，辅助能力有限。如有需要，可以选择 治疗灾厄师 的配置。
  • 使用 炫耀之柱 可以在队友倒地时将他们传送到自己所在的位置。（ 鲜血输送 ）
