直伤增益复制魂兽师

  泰协内容维护小组
  2020.04.06
  未经作者授权，禁止转载

概述

  • 直伤增益魂兽师 是一套野外配置，可以用来跑图 或 做任务。得益于 陆行鸟姿态 可以延长其他技能提供的 急速 和 保护 ，使得该配置在实战中有不错的输出及生存能力。

特性 与 技能

  • 常规
  • 强力生存

特性

  • 永恒联结 ：依然欠缺生存能力，可以考虑点出。

技能

宠物：

  • 输出型宠物（首要宠物）：
    • 幼龄雾鳞蜥 ：依靠 倒地猛扑 获得控制， 使用 烟雾突击 闪避并攻击敌人，也可以利用 直觉交战 、 荒野召唤 、 猛扑 与 障目烟云 触发组合技获得 潜行 。
    • 幼龄岩石瞪羚 ：依靠 甩头猛冲 可以提供控制，其伤害能力要高于雾鳞蜥。
    • 幼龄短面熊 ：可以施放 流血重击 ，属于偏防御型的选择。
  • 保护型宠物（第二宠物，建议携带一个辅助或防御力较强的宠物，并通过 无畏坚韧 或 灵魂舒缓 获得额外的生存能力）：
    • 幼龄蓝花楹 ：合体后提供 灵魂舒缓 、 光合作用 和 召唤闪电 。
    • 幼龄黑熊 ：合体后提供 无畏坚韧 和 蔑视痛苦 。
    • 当处于交战状态无法骑上坐骑时，可以与鸟类宠物进行合体（例如： 幼龄鹰 ），并获得 俯冲 和 急速鸣叫 快速脱战。

精英技能：

  • 独狼之力 ：需要进一步提高输出，可以选用。

特性

  • 永恒联结 ：依然欠缺生存能力，可以考虑点出。

配置说明

技能使用

伤害技能：

  • 你的主要伤害技能是 大地冲击 、 重击 、 双重弧光 和 猎手召唤 。
  • 剑柄猛击 可用于重置 重击 。
  • 使用 直觉交战 和 和风疾行 来获得 急速 。
  • 如果你的主要技能都处于冷却中，只需要保持普攻 以及 在上述技能冷却后使用即可。

防御技能：

  • 该配置可以提供大量的 保护 和 再生 ，可以帮助你抵御怪物的攻击。

  • 烟雾突击 可以用于躲避攻击：

    • 注意，该技能不同于 闪避 的即时生效效果，它有一个延迟性前置动画，如果你想立即使用该技能的闪避效果，请按下“收起武器”快捷键，然后再施放该技能。（设置快捷键：系统设置 - 操作选项 - 其他 - 取出/收起武器）

  • 格挡反踢 可以格挡攻击，如果你格挡时距离敌人过远，将无法触发击退效果，一次格挡可以阻挡多次攻击。

  • 尽可能使用 闪避 或 格挡 躲避敌人的关键性攻击。
  • 如果打怪时依然难以存活，建议点出 永恒联结 ，或使用“生存”特性分支。

说明

增益保持：

  • 使用这套配置的宗旨是尽可能久的维持自己身上的增益，利用特性 速度本质 和 技能 陆行鸟姿态 ，使你更容易保持增益，开启该姿态时，可以利用以下技能叠加增益：
    • 和风疾行
    • 荒野召唤
    • 石牦牛姿态
    • "群攻之力！"
    • 直觉交战
  • 一旦拥有了 威能 、 激怒 、 保护 、 再生 、 急速 ，你就可以利用 "同心治疗！" 将这些增益复制给宠物，然后通过 野兽模式 （ 血肉强化 ）传递给本体，在 20秒后，继续使用 "同心治疗！" 将增益复制给宠物。
  • 在长时间的战斗中，依靠上述小技巧，你可以将增益维持很长时间。
  • 通过 群体领袖 分享姿态，姿态技能本身附带的增益，附近的盟友也能获得，因此在上述的5种增益中，只有 急速 不能被分享。

控制技能（破蔑视）：

  • 剑柄猛击
  • 格挡反踢
  • 倒地猛扑
