直伤刺客预告者

  泰协内容维护小组
  2021.01.22
  未经作者授权，禁止转载

概述

  • 直伤刺客预告者 是一套高爆发的直伤PvP配置，其主要职责是团战，最好可以搭配辅助职业使用，可以弥补该配置相对乏力的续航（主要是弥补症状清除能力）。

特性 与 技能

特性

  • 战斗狂热 ：如果队伍中有两个 直伤预告者 ，可以点出该特性避免重复叠加 刺客灵气 。

配置说明

伤害技能

基本玩法：

  • 该配置的核心玩法是利用游击战术，伺机找寻目标进行秒杀。
  • 直伤刺客预告者 当前版本不再拥有以往较强的续航能力，在没有格挡或闪避时，不宜长时间处于战斗漩涡中。
  • 虽然常规上认为 预告者 是一个近战输出职业，但他有相当多的技能都可以从远处进行施放并对战局走势产生影响。
  • 使用诸如 元素轰击 、 冰冻隔离 、刺客姿态下的 真正本质 、 桎梏波动 从安全距离消耗敌人的血量，冒然进入近战缠斗很容易吃到大量的 AOE伤害。
  • 利用你的控制技能，例如 迷雾激流 或 混乱疾风 从对局中寻找下手的时机，跟进 急速 、 附魔匕首 、 压倒优势 进一步增加爆发能力，让接下来的进攻，每一刻都极具威胁。
  • 桎梏波动 和 死亡打击 是这套配置中的主要爆发技能。
  • 无情突击 在 1v1 时非常重要，因为其伤害不会被额外的目标分摊。在团战中，该技能主要用于闪避攻击。在面对一些有 潜行 能力的敌人时，注意要先使用 黑暗多面体 → 黑暗凝视 为目标施加 现身 ，之后再跟 无情突击 ，因为敌人成功 潜行 后会让该技能中断。

刺客姿态 及 伤害铺垫：

  • 在发起爆发输出前，有很多伤害可以进行铺垫。
  • 附魔匕首 是一个生命虹吸技能，这种虹吸伤害相当于真实伤害，会绕过任何伤害减免措施，如 保护 甚至 痛苦忍耐 。尽管如此，它依然不能无视 闪避 或 无敌（如 失真 ）。
  • 压倒优势 在激活状态下可显著增加其他伤害技能输出能力，但持续消耗的能量比较多，仅在爆发输出的前几秒内激活。注意， 压倒优势 只能追加对单体目标的输出，它可以通过堆叠 强力执念 来增加 AOE 输出技能本身的伤害。
  • 如果你的能量即将耗尽，在达到 0能量前主动关闭 压倒优势 只会让该技能进入 1秒的CD，若能量耗尽让其自行关闭，则会进入 4秒的 CD。
  • 利用 传奇刺客姿态 下的 真正本质 （ 自然多面体 第二段）触发 龙吟回响 后，会让 自然多面体 的被动持续几秒。
  • 位面穿行 非常适合在快速切入战局，在获得 急速 的同时，还能使接下来的两次攻击不可被格挡。施加的 急速 有助于施展爆发技能 或 是直接使用普攻将目标血量压低，而不可被格挡的特质，也将那些企图利用格挡垂死挣扎的敌人，打入倒地状态。
  • 翠玉风暴 利用施放假动作，欺骗敌人 闪避 是最常用的手法。在 翠玉风暴 读条时会出现巨龙猛击双翼的动画，此时使用 “取出 / 收起 武器” 快捷键即可中断施法，并使该技能进入 4秒CD。由于该技能消耗能量 35点，建议在关键情形下再使用，例如 帮助队友创造击杀机会，或是中断敌人的 复活 或 终结引导。

远程消耗连招示例：

  1. 附魔匕首
  2. 位面穿行
  3. 压倒优势
  4. 桎梏波动 （单手剑4）
  5. 武器其他技能 或 切换至法杖使用 5技能 迷雾激流

近战秒人连招示例：

  1. 混乱疾风
  2. 可选 - 切换至 传奇刺客姿态 激活 压倒优势
  3. 桎梏波动 （单手剑4）
  4. 死亡打击 （单手剑5）
  5. 冰冻隔离 （单手剑2）

续航 及 清症

  • 圣光多面体 → 圣光灌注 可以在使用后将受到的伤害全部转化为治疗，并获得无敌，这是应对自身被集火时最好的技能。确保你至少能从该技能开启时获得 10k的治疗。小心在开启后敌人对你放弃攻击，在开启 圣光多面体 后，等待几秒再使用 圣光灌注 通常可以避免该情况发生。 圣光多面体 有读条，而 圣光灌注 是瞬发的，并且可以在昏迷状态下使用。
  • 激活 混沌多面体 可通过 抗性符文 获得 抗性 。
  • 法杖 + 传奇刺客姿态 可以提供极强的逃脱能力， 暗影回撤 提供 15点耐力，并能解除昏迷、移除限制移动类症状，除此之外，法杖技能 裂隙结界 和 迷雾激流 还能提供短暂的 格挡 和 闪避。
  • 当你需要使用 附魔匕首 获得治疗时，最安全的办法就是在 裂隙结界 、 迷雾激流 、 无情突击 的 格挡 或 闪避 过程中使用。

直伤刺客预告者 不能应对大量的症状，但有数个小技巧可进行小型清症：

  • 一整套清症连招可以清除掉 8种症状（如果计入 目盲 那就是 9种症状，通常你应该在清除症状前移除掉 目盲 ）
  • 理想状况下可以使用 暗影回撤 摆脱所有的硬控症状（如 恐惧 、 嘲讽 ），仅此一个技能就可以移除 4种非伤害型症状，这样可以利用其他技能单独清除伤害类症状，同时，该技能附带的闪避效果也可以防止短时间内受到其他症状。
  • 切换到法杖可以通过触发 洁净法印 移除一个症状，切换姿态可以触发 净化引导 移除一个症状。如果你已经切换至法杖，切换姿态即可触发 洁净法印 。
  • 复原波动 （法杖4）移除两个症状。
  • 在这之后一段时间内，你应该安全无忧，但这样做也耗尽了你的能量，你需要防御一段时间，通过格挡 和 闪避 防止被重新堆叠症状，最好可以退回到安全区域修整片刻。

说明

  • 切换姿态可以将能量恢复至 50点，移除一个症状（ 净化引导 ），并触发各种特性获得增益（如： 唤醒折磨 、 灵魂增益 ），还能触发法印的切换武器效果（如果不处于CD中）。
  • 切换姿态是能量管控方法中重要的一环，尽量在能量见底或者主要技能都进入CD后进行切换。

  • 开局时切换至 传奇巨龙姿态 ，在基地中开启以下技能为队友叠加增益：

    • 元素多面体
    • 力量多面体
    • 黑暗多面体
    • 自然多面体 （在即将开门时激活它，特别是它可以延长队友施加的增益，例如：游侠的 荒野召唤 ）

  • 在战斗中，尽量不要将 传奇巨龙姿态 下的通用技能开启超过 3秒。建议只激活 元素多面体 和 力量多面体 ，然后立即使用 元素轰击 和 力量爆发 来制造可观的 AOE伤害。

  • 混乱疾风 可以将已经倒地的敌人击飞，可以利用这点将他们从争夺点上推出去，这可以在占据争夺点的同时，延缓它们的生命恢复。
  • 两种姿态下的精英技能都有明显的施法动画，如果在读条时，敌人对你施加了 目盲 ，立即使用 “取出/收起武器” 快捷键取消施法。当然，你也可以利用这一假动作欺骗敌人施放 闪避 或 格挡技能。

  • 黑暗多面体 是该配置中最灵活的技能：

    • 黑暗凝视 是一个低消耗的解除昏迷技能，并且可以为敌人施加 现身
    • 黑暗凝视 可以施加 目盲 限制敌人施放强有力的技能，从而避免浪费 闪避 ，并且在终结倒地敌人时（战士 / 魂武 / 工程 / 游侠 / 唤灵），使他们 目盲 以避免终结引导被打断，而施加的 现身 也很容易阻止倒地的 幻术师 使用倒地技能进行隐身。
    • 如果 你在施放 混乱疾风 前被敌人施加了 目盲 ，可以在 混乱疾风 即将施放完毕前，使用 黑暗凝视 作为垃圾技能抵消掉 目盲 ，让 混乱疾风 可以被安全施放。

  • 在非战斗状态下，移速增加上限为 33%，此时 迅捷 和 奔驰疾速 将拥有相同的移速。在战斗中， 压倒优势 将提高你的移速，这使你更容易脱战。

  • 传奇刺客姿态 下的 真正本质 （ 自然多面体 第二段 ） 是一个优异的 AOE 伤害技能。

被克制：

  • 高压制力的症状职业
