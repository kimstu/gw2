. 德鲁伊 . Druid

治疗德鲁伊

  暴君之刃、落雨丶凉城 [InS]
  泰协内容维护小组
  2021.01.22
  未经作者授权，禁止转载

概述

  • 治疗德鲁伊 是学习其他诸多职业配置的起步配置。它易于上手，但如果你是第一次玩 治疗德鲁伊 ，一些基础知识必须掌握。推荐开荒 或 路人队伍使用。
  • 与所有的 德鲁伊 配置相同，本配置除了作为团队中最主要的 威能 提供者，还可供应大量的进攻型 BUFF，例如各类精魂（ 冰霜精魂 、 旭日精魂 ）、 眼观六路 以及多种增益，并且战宠 和 天界化身 状态下可以提供大量的破蔑视技能。由于其强力的辅助能力，因此 德鲁伊 成为大型地下城模式必不可少的配置之一。
  • 治疗德鲁伊 的治疗能力在大多数情况完全够用，但与 治疗暴风使 这样专注于治疗的职业相比， 治疗德鲁伊 更多是充当 进攻型 BUFF 提供者的角色。
  • 治疗德鲁伊 非常适合初学者使用，请详细阅读以下指南，学会根据实际情景选择搭配技能及战宠。了解技能运作、丰富实战经验将使你更好的掌握该配置。一旦你及你的队伍有了更多实战经验，可以尝试使用速刷配装。

特性 与 技能

  • 推怪（双德阵容）
  • 速刷（单德阵容）

特性

  • 荒蛮反射 ：不使用陷阱技能时，点出该特性。
  • 陷阱专长 ：治疗技能使用 治疗之泉 时推荐点出。
  • 盟友救助 ：推荐 开荒 或 路人队伍 点出，以提升容错。
  • 直觉反应 ：推荐 固定队 或 有经验的路人队伍使用。
  • 活力降临 ：若武器不携带战号，或是实战中缺乏 激怒 和 活力 ，推荐点出。
  • 翠绿蚀刻 ：选用雕文技能时点出。
  • 有经验的队伍可以使用双 蔑视 宠物。

技能

治疗技能（可选）：

  • 流水精魂 （ 水流涌动 ）：可以通过攻击，长时间获得小额治疗，在伤害不致命且比较均匀的对战中，可以选择使用（例如戴莫斯拉黑水时）。主动使用 水流涌动 可以提供中等程度的治疗 和 再生 。
  • 治疗之泉 ：更广的症状清除范围，持久的流水组合技区域， 以及过剩的再生，将更快为天界之力充能。使用该技能时，需要点出特性 陷阱专长 。
  • 回春雕文 （ 回春雕文 ）：通常作为爆发型治疗技能使用，治疗范围广，天界状态下对队友进行高额治疗，对自己及宠物进行小幅治疗。

通用技能（可选）：

  • 冰霜精魂 （ 寒流涌动 ）：用于提升团队 5%的直伤伤害，主动使用 寒流涌动 可以提供 3层 威能 。
  • 旭日精魂 （ 烈日耀斑 ）：使队友的攻击可以堆叠 燃烧 ，从而提升团队整体的症状伤害，症状队必带。主动使用 烈日耀斑 可以提供 活力 ，并能给周围的敌人施加 目盲 和 2层 燃烧 。
  • 风暴精魂 （ 召唤闪电 ）：使队友的攻击可以堆叠 易伤 。主动使用 召唤闪电 可以提供 激怒 ，并能制造 1秒的 晕眩 用于破蔑视。
  • 磐石精魂 （ 流沙陷阱 ）：使队友的攻击可以为自身叠加 保护 。主动使用 流沙陷阱 可以提供 保护 ，并能 残废 和 定身 附近的小怪。
  • 潮汐雕文 （ 潮汐雕文 ）：是一个短CD的击退技能，可以用于推怪或者破蔑视，天界状态下使用会将附近的小怪拉向自己。
  • 平等雕文 （ 平等雕文 ）：需要破蔑视 或 小范围眩晕小怪时，可以选用。
  • 开始搜救 ：将倒地后难以施救的队友拉到自己所在位置。
  • 快保护我 ：获得保护，解除队友的昏迷状态，通常用来应对某些 BOSS 的恐惧。
  • 泥泞地表 ：需要控小怪时，选择使用。
  • 磐石纹章 ：激活后可以让自己短时间内处于直伤免疫状态，通常在戴莫斯拉黑水时使用。

精英技能（可选）：

  • 自然精魂 （ 自然复生 ）：主动使用将自然精魂移动到自己所在位置，若此时附近有队友倒地，自然精魂会牺牲一部分生命值来复活队友。只有使自然精魂移动后，它才能启动复活操作。主动使用 自然复生 可以提供 稳固 ，并能移除5名队友的 3个症状。
  • 群星雕文 （ 群星雕文 ）：持续时间内避免添加新的症状，并脉冲式清除症状，通常需要提前预判施放。
  • 藤蔓缠绕 ：范围型控怪技能。

注意：

  • 精魂类主动技能提供的增益，必须在点出 自然之仇 时才能生效。
  • 大团 治疗德鲁伊 携带 旭日精魂 、 磐石精魂 并使用主动技能叠加增益或伤害，脱离大团的机制位 治疗德鲁伊 携带 冰霜精魂 、 风暴精魂 。

特性说明

  • 在对抗特定的 BOSS 时，你可能会需要 长弓4 近身平射 穿透目标，射手特性分支中的 顺风动势 将提供这种能力。
  • 醉生梦死 还能提升 召唤闪电 （ 风暴精魂 主动技能）和 月色引力 的破蔑视能力。

特性

  • 岩石形态 ：推荐开荒 或 生存环境恶劣 时点出。
  • 盟友救助 ：推荐 开荒 或 路人队伍 点出，以提升容错。
  • 翠绿蚀刻 ：选用雕文技能时点出。

配置说明

起手

确保天界能量为满状态，如果不满，请求队友 /gg 然后手动复活他获得天界能量。
冰霜精魂 、 旭日精魂 在战斗开始前放于战斗边缘位置。
未进入战斗前，不要急于刷 威能 ，否则进入战斗后会被清除。

  1. 寒流涌动 、 烈日耀斑 将两个精魂移动至战斗发生位置，并依靠 自然之仇 堆叠 威能
  2. 猎手召唤
  3. 荒野召唤
  4. 天界化身
  5. 回春潮汐 （触发 疾速箭射 ）
  6. 月色引力
  7. 回春潮汐
  8. 终止天界化身形态。

注意：依靠 大地恩泽 获得的 威能 是持续时间最久的，多余的 威能 会自动选择替换掉持续时间最短的 威能

爆发治疗

  1. 天界化身 （触发 疾速箭射 ）
  2. 月色引力
  3. 治疗之泉 或 回春雕文
  4. 回春潮汐
  5. 生命种子
  6. 月色引力
  7. 终止天界化身形态。
  8. 朝残血队友位置施放 崇高转化
  9. 朝向 崇高转化 的位置使用 先辈恩泽 并触发组合技

天界化身

  • 天外射线 是最低优先级的技能，可以使用它去填补其他技能进入冷却的空隙。
  • 生命种子 有一个短暂的施法时间，尽可能在其他主要治疗技能进入冷却时使用它。症状移除可以帮助团队应对可能遭受到的症状伤害。必要情况下，可以使用其 目盲 效果，干扰一些带控制的小怪（例如某些会踢人的小怪）。
  • 月色引力 施法快且治疗量高，也可以用来破蔑视，甚至需要爆炸组合终结技时使用它。
  • 回春潮汐 是最容易为团队叠加 威能 的技能，它同时也能产生一个流水组合技区域。
  • 自然汇流 有很好的控制效果，必要时可以用来 缓速小怪 或 破蔑视。
  • 实战中应注意，如果团队不是非常缺乏治疗，应在叠加 威能 后尽快退出天界状态，以保留一部分能量在必要时使用。

天界之力获取方式：

  • 治疗盟友 或 对敌人造成伤害。
  • 触发 翠绿蚀刻 并影响队友。
  • 在你血量较高的状态下使用 回春雕文 ， 并触发 高等协同 。
  • 在使用法杖时， 日光射线 是你产生 天界之力 最快的方法。

说明

威能：

  • 荒野召唤 （战号5）可以为队友叠加 6层 威能 。
  • 回春潮汐 和 自然汇流 的每一次脉冲可以叠加两层 威能 ，其他天界技能单次施放只能叠加两层 威能 。
  • 雕文类技能依靠特性 翠绿蚀刻 触发的 次级生命种子 可以叠加两层 威能 。
  • 点出特性 自然之仇 后，使用 寒流涌动 （ 冰霜精魂 ） 可提供 3层 威能 。

再生：

  • 治疗之泉 除直接治疗外，还能提供 再生 。
  • 点出特性 自然之仇 后，使用 水流涌动 （ 流水精魂 ）可提供 再生 。
  • 点出 乘风音律 后，使用战号技能可获得 再生 。
  • 崇高转化 （法杖5）
  • 流水 + 投射物 组合技
  • 幼龄蕨犬 （ 再生之音 ）
  • "开始搜救！"

激怒：

  • 荒野召唤 （战号5）
  • 点出特性 自然之仇 后，使用 召唤闪电 （ 风暴精魂 ）可提供 激怒 。
  • 点出特性 活力降临 后，可依靠切换宠物获得 激怒 。
  • 宠物携带 幼龄猛虎 时，可依靠宠物技能 暴怒猛扑 全程覆盖 激怒 。

保护：

  • 点出特性 自然之仇 后，使用 流沙陷阱 （ 磐石精魂 ）可提供 保护 。

活力：

  • 点出特性 自然之仇 后，使用 烈日耀斑 （ 旭日精魂 ）可提供 活力 。
  • 点出特性 活力降临 后，可依靠切换宠物获得 活力 。

稳固 和 解控：

  • 点出特性 自然之仇 后，使用 自然复生 （ 自然精魂 ）可提供 稳固 。
  • "快保护我！"

治疗：

  • 法杖的 日光射线 和 星界灵火 是对抗环境伤害最主要的小幅治疗技能，天界状态下的 回春潮汐 和 月色引力 是重要的爆发型治疗技能。
  • 流水 + 爆炸 组合技可触发一次群体治疗，例如：最常用的就是法杖技能 崇高转化 + 先辈恩泽
  • 进入天界化身时，可以触发特性 疾速箭射 ，通常会用它来加快 月色引力 的冷却。
  • 回春雕文 在天界状态下可提供高额治疗（ 回春雕文 ）。

清症：

  • 治疗之泉 可清除两个症状。
  • 群星雕文 在持续施法期间，可持续性清除症状。
  • 光明 + 爆炸 组合技可触发一次群体群体清症（清除一个症状）。
  • 生命种子 （天界技能2） 可清除一个症状。
  • 点出特性 翠绿蚀刻 后，使用雕文技能，依靠 次级生命种子 可清除一个症状。
