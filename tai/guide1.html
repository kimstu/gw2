迷雾碎层入门指引

  暴君之刃
  2019.11.15
  未经作者授权，禁止转载

简介

  • 迷雾碎层 是一种特殊的 5人地下城模式，每一层都有独立的 副本剧情和场景。
  • 迷雾碎层 是提升装备的重要方式，并能提供 灌注升级 以及 灌注槽材料。

进入碎层

碎层入口：

  • 狮子拱门 - 马里纳堡垒传送点 [&BDAEAAA=]，南边。

  • 雾锁庇护所 东侧，需要 雾锁庇护所通行钥匙 或 雾锁庇护所通行钥匙（2周） 才能进入。
  • 当小队成员在 狮子拱门 或 雾锁庇护所 时，同地图的任意一名玩家进入 迷雾碎层 都会提示其他玩家是否进入。
  • 进入 碎层入口传送门 时，会直接提示触碰的玩家选择碎层难度。

雾锁观测台（碎层大厅）

  • 进入 迷雾碎层 后，左边能看到 难度设定排序器，可以使用它调整碎层难度。

  • 当碎层难度设定完毕后，一名玩家跳入入口前方的传送盘内，就会提示其他进入 迷雾碎层 的小队成员 是否准备好继续了？。

  • 碎层入口处有一个 FIX-0255型 修理NPC，每完成一层碎层后，可以访问它出售垃圾道具，并维修装备。

  • 可以使用小型蓝色传送盘在大厅内的三个浮岛间快速穿梭。

碎层内有各种商人：

  • 灌注魔像5959 出售各种与灌注相关的物品。
  • BUY-2046 PFR型 出售粉色升华戒指 以及 迷雾首饰。
  • BUY-4373型 出售收藏物品 和 升华拆解工具、呼吸器图纸、 碎层解密钥匙 。
  • 黛萨的炼金师 出售或回收迷雾药剂。
  • BLING-9009 出售无尽迷雾药剂 和 升华装备 图纸、皮肤、 天界灌注（红色） 。

与普通五人地下城区别

  • 任何从迷雾碎层大厅外进入 碎层关卡副本 的玩家都会阵亡。
  • 80级以下玩家进入副本会进行等级缩放，允许低等级的玩家参与战斗，但从装备中获得的属性不会缩放提高，并且副本中的怪物会优先攻击低等级玩家。

难度层级

  • 与普通 5人地下城不同，迷雾碎层 采用难度层级模式。
  • 每当你完成 难度层级 等于 或 高于 个人难度层级的碎层时，你的 个人难度层级 会提升一级，直到满级 100级为止。
  • 在迷雾碎层大厅中，可以通过 难度设定排序器 变更当前要加入的碎层层数。
  • 碎层的战利品丰厚程度与当前加入的 难度层级 有关，即使你的个人难度层级较低，加入高难度也会获得对应等级的奖励。
  • 碎层设置 面板列出四个大段难度等级，未满 80级的角色将不能选择难度，80级时会解锁 等级:1 的25层，只有个人层级等级接近每个大段的最高级时，才能解锁下一个大段。
  • 个人难度层级 全角色通用。

更高难度层级的变化

  • 同一地图的高层版本，敌人产生的数量更多，等级也更高，并且敌人的生命值也增多，伤害能力也加强。
  • 个别的精英怪 和 首领会获得新技能。
  • 在 25、50、75层后将逐步出现更多机制变化。
  • 从 20层开始，痛苦伤害 被加入到小怪 和 BOSS的攻击中，并且碎层层级越高需要的痛苦抗性就越多。
  • 从 26层开始，为副本添加一种 雾锁异变 状态，在 51级时添加两种，76级时添加三种。
  • 从 71层开始，玩家被打入倒地状态，碎层复仇者 会出现并试图终结倒地玩家。
  • 碎层复仇者 在终结时被打断或杀死，就会消失。倒地玩家利用 隐身 或 闪烁 位移到足够远的距离，也能摆脱 碎层复仇者。
  • 如果遇到 雾锁异变:碎层捍卫者，碎层复仇者 就会被替换为 碎层捍卫者，它会持续攻击玩家直到被杀死为止。
  • 当一名倒地玩家对应的 碎层捍卫者 未被击杀时，该玩家连续倒地不会刷出新的 碎层捍卫者。

雾锁奇点

  • 碎层副本中途没有 传送点，取而代之的是使用 雾锁奇点 作为进度存档点，如果一名小队成员阵亡，必须等待其他玩家都脱离战斗后才能从 雾锁奇点 存档点位置处复活。
  • 如果队伍中有一名玩家提前跑到了下一个 雾锁奇点 位置处，其他玩家可以输入命令 /gg 快速自杀后，即可直接复活到新的 雾锁奇点 存档点位置处，以便节省跑路时间。

  • 在拥有 碎层协调：雾锁奇点 专精后，你才能使用以下能力：

  • 与 雾锁奇点 互动可以增加 5点 痛苦抗性。
  • 与 雾锁奇点 互动可以刷新技能冷却，一般情况下，辅助职业可以在与 BOSS 交战前，先为队友刷满增益，随后使用 雾锁奇点 刷新技能，再开启战斗。
  • 碎层副本中途没有 修理点，如果玩家掌握 碎层协调：雾锁奇点 专精，可以与 雾锁奇点 互动来修理装备。
  • 掌握 碎层协调：雾锁奇点 专精，与 雾锁奇点 交流会获得一层特殊的护盾，在受到致命伤害后，会将护盾消耗并获得约 0.5秒的无敌，并将生命值恢复为 25%的最大生命值。

  • 玩家未脱战时，无法与 雾锁奇点 交流。
  • 雾锁奇点 被交流后，只有队伍中的所有玩家都阵亡后，雾锁奇点 才会重置为可交流状态。

雾锁异变

  • 雾锁异变 是一类特殊状态，它会在副本过程中对玩家产生特殊影响。
  • 从 2级碎层开始（26层）开始，副本中会开始添加一种 雾锁异变，3级（51层）开始添加两种，4级（76层）开始添加三种，雾锁异变 每层每天都是随机分配的。

火焰之血 敌人受到伤害时制造烈焰飞弹。受到的症状伤害减少20%。（冷却时间：2秒）

痛苦

  • 玩家在20层碎层会首次遇到 痛苦 ， 痛苦 会减少 70%的 治疗量 和 屏障厚度。
  • 痛苦 每3秒按照角色最大生命值百分比造成一次伤害，并且随着碎层层级提升而进一步增加。
  • 与 症状伤害 不同，在玩家倒地后 痛苦 不会消失，依然会受到伤害。
  • 部分敌人的攻击也会施加 痛苦 ，可以依靠闪避、格挡、无敌规避这些攻击。
  • 部分碎层在与最后的BOSS遭遇前，会进行 痛苦抗性检查，此时所有玩家都会受到 痛苦伤害。
  • 下表列出 痛苦伤害 与 痛苦抗性 的伤害减免关系。

痛苦抗性

  • 可以通过提升 痛苦抗性 将 痛苦伤害 降低到最少 1%。
  • 痛苦抗性 依靠安插在升华或传奇品质装备上的 痛苦灌注 提供，与副本中的 雾锁奇点 交流可以暂时提高 痛苦抗性。
  • 通过 账号强化：痛苦阻抗 以及 账号强化：迷雾协调 每级可以获得 5点 痛苦抗性。可以在 碎层大厅 找到 德洛伊尔 进行账号强化。

  • 下表列出了 碎层设置 中每层建议的最低抗性值。
