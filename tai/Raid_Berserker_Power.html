直伤狂战士

  TheCryophoenix [SC]、南巷挽清风 [XF]
  泰协内容维护小组
  2021.01.20
  未经作者授权，禁止转载

概述

  • 直伤狂战士 是一套拥有高爆发、良好持续输出、中等AOE伤害能力的直伤配置。
  • 该配置需要确保 敏捷 的良好覆盖，以保证 狂暴 可以提供最大化输出。
  • 直伤狂战士 除了破蔑视可以用到 头槌重击 外，不能为团队提供其他辅助功能，因此请根据团队需求选择使用。

说明

  • 开场结束后，技能将基于一定的优先级施放，循环的主要目标是尽可能延长狂暴状态的持续时间。
  • 当处于 狂暴 状态时，你总是期望频繁的使用 斩首 ，为了达到这一目的，你可以通过触发三次打击 或是 其他方法快速堆满怒气，以下按优先级顺序全部列出：
  1. 头槌重击 + 怒气涌流
  2. 鲜血清算
  3. 旋风斧斩 + 单手斩 或 飞斧投掷
  4. 双重打击 + 单手斩 或 飞斧投掷
  5. 单手斩 或 飞斧投掷 + 切换武器 （ 通用怒火 ）
  6. 回旋斧斩
  7. 单手斩 + 双重斩
  • 在 狂暴 状态之外时，不要打断自动攻击链。
  • 头槌重击 → 怒气涌流 冷却即用，你可以在 头槌重击 读条过程中就使用 怒气涌流 。
  • 怒气涌流 是瞬发技能，主要用于延长狂暴持续时间。
  • 由于 怒气涌流 解除 头槌重击 的昏迷后可以减免一半冷却，因此在 头槌重击 → 怒气涌流 使用完毕后（此时 头槌重击 会进入 20秒CD），你应该在狂暴状态中再使用一次 怒气涌流 。
  • 蛮力打击 冷却即用。
  • 在个别情况下可以装备一把巨剑，以提高机动性和些许起手爆发输出，但你将失去将 切换武器 整合进输出循环获得怒气的能力。
