症状暴风使

  泰协内容维护小组
  2020.06.28
  未经作者授权，禁止转载

概述

  • 症状暴风使 适合单兵作战的情景，这套配置有非常高的症状输出能力 和 不错的生存能力。

特性 与 技能

特性

  • 致盲灰烬 ：对抗群怪时，可以依靠该特性给予小怪 目盲 。

技能

治疗技能（可选）：

  • 元素和谐雕文 ：爆发性治疗。
  • 幻光恢复 ：清症。
  • 秘法绚爆 ：爆炸终结技。

通用技能（可选）：

  • "极寒冰结！" 或 "天崩地裂！" ：团队辅助技能，为附近队友提供防御。
  • 风暴雕文 ：可以使用 火焰风暴 制造AOE输出，也可以使用 沙尘风暴 施加 目盲 。
  • 秘法护盾 ：解除昏迷，格挡攻击。
  • 电光火石 ：闪烁类位移技能。
  • 元素能量雕文 ：如果你觉得召唤的小型元素使你很难脱战，可以使用该技能替换。

精英技能（可选）：

  • 炽焰巨剑 ：高额的AOE伤害但冷却时间较长，也能提供一些机动能力。

装备 与 消耗品

  • 生存（入门）
  • 极限输出
  • 终极生存

装备说明

  • 生存配装使用 恐怖 属性，尽管装备获取成本较为低廉，但输出能力却是三套装备中最差的，可以作为入门过渡装备使用。
  • 恐怖 属性可以直接从普通升华装备盒中开出。

配置说明

技能使用

伤害技能：

  • 你的主要伤害技能是： 炽焰疾步 、 荒火哀嚎 、 龙蜥吐息 、 尖刺石环 ，通用技能尽量保持冷却即用。
  • 闪电宝珠 可以对大体积目标造成高额伤害。
  • 上述技能进入冷却的间隙使用 巨龙之爪 或 通体穿刺 。

防御技能：

  • 频繁施放技能可触发 再生纹章 从而获得小额治疗。
  • 流水过载 是一个强力防御型治疗技能，与 潮汐激荡 或 流水宝珠 连同使用以获得更强的治疗。
  • 切换至 流水协调 时， 冰封爆裂 与火焰区域触发组合技可获得 威能 ， 冰封爆裂 与 流水宝珠 触发组合技可获得额外的治疗。即使配装中没有治疗效果属性，爆炸与流水区域触发的组合技也能提供高额治疗。
  • 砂石狂啸 可以提供 保护 ，除此之外还能提供反射投射物、延长增益持续时间的能力。
  • 沙尘风暴 在对抗大波敌人时，施加 目盲 有奇效。
  • 过载技能均可解除昏迷。

威能施加：

  • 火焰过载 ：在读条过程中它就能源源不断的为走入其内的队友提供 威能 ，施法完毕后会在地上留下一片火焰区域，并为进入其中的队友继续提供 威能 。最终可提供 10层 威能 （约16秒）。
  • "炽火燎原！" ：3层（10秒）。
  • 火焰区域（ 火焰过载 、 炽焰疾步 、 荒火哀嚎 ） + 爆炸终结技（ 冰封爆裂 、 "天崩地裂！" ），每次触发可提供 3层 威能 （20秒）。

说明

常规：

  • 施放 龙蜥吐息 时注意不要被其他技能打断，切换协调时该技能不会被打断。
  • 交战中，尽量保持 火焰纹章 、 "炽火燎原！" 、 小型元素雕文 冷却即用。
  • 在进行具有挑战性的交战前，务必提前施放 元素雕文 。
  • 元素雕文 在 火焰协调 下召唤出的火元素伤害输出较强，在 大地协调 或 流水协调 下召唤出的土元素或水元素具有较强的防御能力。
  • 冰封爆裂 或 砂石狂啸 与 火焰区域触发组合技可获得 3层 威能 。
  • 元素雕文技能冷却即用。
  • 热力同步 尽量在你拥有高层数的 威能 时使用。

控制技能（破蔑视）：

  • 空气过载 / 大地过载
  • 大地突刺
  • 冰封爆裂
  • 潮汐激荡
  • 气流漩涡
  • 电击光环 → 逆向转化闪电
