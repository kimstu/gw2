直伤纹章战士

  暴君之刃
  2020.04.06
  未经作者授权，禁止转载

概述

  • 直伤纹章战士 是一套非常流行的新手野外配置，该方案没有繁琐的技能操作，新玩家很容易上手。

特性 与 技能

特性

  • 强力正义 ：需要提升治疗能力，并且搭配 超级力量法印 时可使用。
  • 碎裂射击 ：配备步枪时使用。

技能

治疗技能（可选）：

  • 伤势愈合 ：治疗能力略强于 治疗纹章 ，可用于清症。
  • "突破极限！" ：更强力的瞬时恢复技能，使用时可以填满怒气值。

通用技能（可选）：

  • 耐力纹章 ：被动提升耐力恢复速度，这将提高使用闪避的频率。
  • "重整旗鼓！" ：如果症状压力较大，可以用于替换 牦牛纹章 。
  • 痛苦忍耐 ：短时间内免疫直伤伤害。

精英技能（可选）：

  • 怒神附体 ：需要额外的控制技能，可以选择使用。

装备 与 消耗品

  • 标准
  • 极限输出
  • 强力生存

配置说明

技能使用

伤害技能：

  • 你的主要伤害技能是 旋风斧斩 、 双重打击 和 回旋斧斩 ，在这些技能冷却时，优先使用它们。除此之外，尽可能使用斧的普攻打输出。
  • 不要打断斧1的自动攻击链，普攻的大部分伤害都来源于第三段 三连斩 。
  • 剖脏 冷却即用。
  • 需要打爆发时，激活 盛怒纹章 以获得提升输出的增益。
  • 在短期战斗中，可以激活 威能纹章 和 怒气纹章 提升伤害，在战斗中使用纹章还可通过 纹章专精 获得增伤BUFF。
  • 闪避 到敌人脸上，通过 野蛮闪避 可以伤害敌人并获得 威能 。

防御技能：

  • 尽可能使用 闪避 躲避敌人的攻击。
  • 切换武器 可以让你清除症状（ 混斗复元 ），如果遭遇更严重的症状伤害，请携带技能： 伤势愈合 或 "重整旗鼓！" 。
