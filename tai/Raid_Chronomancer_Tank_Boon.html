. 时空术士 . Chronomancer

坦克增益时空术士

  暴君之刃
  泰协内容维护小组
  2021.01.22
  未经作者授权，禁止转载

概述

  • 坦克增益时空术士 是一套通用性非常强的辅助配置。
  • 坦克增益时空术士 的配置主要为了针对大型地下城中以坚韧值最高为仇恨目标的BOSS使用的，若作为队伍中的第二名时空术士使用，可以适当降低坚韧值。
  • 在大型地下城中，需要与另一名 增益时空术士 或 急速燃火者 和 敏捷预告者 配合，才能使整队100%覆盖 急速 和 敏捷 。
  • 除此之外， 增益时空术士 还能提供 圣盾 来抵御关键性的攻击，并且有非常强的破蔑视能力。
  • 注意， 增益时空术士 在当前版本主要作用是延长队友身上的增益时间，因此在对抗个别BOSS时，可以在阵容中搭配 潜行者 的增益类配置，以获得多种增益来源。

特性 与 技能

  • 混沌 灵感
  • 幻象 灵感

特性

  • 万华领悟 ，增加隐身时间，携带 群体隐形 时点出。
  • 聚能反馈 ，需要反射投射物时点出。

技能

治疗技能（可选）：

  • 永恒法池 ：范围性治疗技能，并附带群体症状清除能力，使用该技能更容易保持队友满血，以使他们的 超级学者符文 增伤效果常驻。
  • 幻光纹章 ：属于进攻型治疗技能，适合搭配输出装备使用，用以加快幻象生成速度。

通用技能（可选）：

  • 灵感纹章 ：常规选择，通常不会被其他技能替换，用于延长队友增益持续时间。
  • 镜像复制 ：用于复制 灵感纹章 延长增益。
  • 回想法池 ：可以为队友提供 敏捷 。
  • 行动法池 ：可以为队友提供 急速 。
  • 预知法池 ：可以为队友提供 圣盾 和 稳固 ，通常在输出环境恶劣时携带。
  • 斥力穹窿 ：用于反射投射物，以应对部分 BOSS 的机制（如：马蒂亚斯）。幻光区域 + 物理投射物 / 旋风 可造成额外的 困惑 。
  • 幻影除魔 ：用于去除目标身上的增益。

精英技能（可选）：

  • 时间扭曲 ：常规选择，用于为队友提供 急速 。
  • 重力陷阱 ：需要更多的破蔑视技能时，可以选用。
  • 谦逊纹章 ：最强的破蔑视技能，对已被变鸟的敌人再使用，不能延长生效时间，如果队伍中的其他 幻术师 也携带了该技能，请协商好施放顺序。
  • 群体隐形 ：一般在灵魂山谷的公墓事件时使用。

特性说明

  • 使用 幻象 特性分支将使你更快的生成分身，降低粉碎技能的冷却时间，并且 F1 技能 分裂时间 将拥有两层充能。

特性

  • 聚能反馈 ，需要反射投射物时点出。

配置说明

增益提供

急速 （覆盖量在通常情况下都是溢出的）：

  • 时间潮汐 ，推出的能量潮可以叠加 急速
  • 行动法池 ，3秒脉冲效果结束后提供
  • 时间扭曲
  • 粉碎技能

敏捷 ：

  • 时间潮汐 ，收回的能量潮可以叠加 敏捷
  • 回想法池 ，3秒脉冲效果结束后提供

圣盾 ：

  • 特性 群体失真 和 模糊铭文 使得施放 纹章 类技能时可以为队友提供 圣盾 ，除此之外，F4 技能 时空裂缝 也可以使你获得 失真 。
  • 预知法池 ，3秒脉冲过程中就能提供

技能使用

  • 使用上述技能时，你必须同时覆盖自己与队友，否则有一方将不能获得增益，因此，团战的抱团站位非常重要。
  • 除此之外，还需要使用 灵感纹章 延长小队成员身上的增益持续时间，注意该技能不会为队友添加未拥有的增益。如果点出 模糊铭文 （灵感 3 - 3），该效果将作用于 10名玩家。
  • 在增益持续时间不足时，你需要携带 镜像复制 来复制 灵感纹章 ，以进一步延长增益覆盖时间。
  • 当与增益覆盖相关的技能都处于冷却中时，你只要保持正常攻击即可，如使用自动攻击链、施放 恍惚狂热 、保持幻象攻击等。
  • 在拥有三个分身后， 分裂时间 冷却即用。
  • 如果分身不足，则用 幻光分身 （节杖1第三段）、 幻象跳跃 制造分身。
  • 在 分裂时间 进入冷却后，再使用 回归原位 。
  • 在没有其他分身时，使用 幻象跳跃 制造分身。

时空裂缝

  • 在 时间潮汐 读条过程中施放 时空裂缝 。
  • 在 时空裂缝 生效的时间内使用的技能，会在 时空转移 后重置冷却，尽可能在开启后多使用覆盖增益的技能。
  • 时空裂缝 快冷却完毕时，其他主要的增益技能要延后到 时空裂缝 开启后再使用。
  • 每一个幻象都将延长 时空裂缝 的生效时间，注意开启前保留幻象，必要的话，可以使用 幻象跳跃 来补满 3个幻象。
  • 时空裂缝 在结束后，会将你传送至开启时的位置，在对抗一些 BOSS 时，这有可能会导致你阵亡。（如：末世魔监工）
  • 时空裂缝 另一个用途是重置 时间扭曲 或 谦逊纹章 的冷却时间，这两个技能有非常长的冷却时间。
  • 在对抗一些 BOSS 的时候， 对破蔑视速度要求非常快，这种情况下，应尽可能在 时空裂缝 生效时施放 重力陷阱 或 谦逊纹章 。（如：马蒂亚斯，萨玛洛格）
  • 如果 时空裂缝 还有很长时间才能冷却完毕，你可以使用粉碎技能来增加输出。
  • 时空裂缝 无法为 镜像复制 刷新冷却。

说明

坚韧检查：

如果你作为主T，对抗“坚韧”仇恨的 BOSS 时，请检查你的坚韧是否为团队中的最高值，以下 技能 或 特性 会在实战中增加坚韧，请基于下述标准计算最低“坚韧”阈值：

  • 120： 编织者 - 元素复调
  • 150： 魂兽师 - 兽群头领
  • 300： 燃火者 - 勇气圣书 的 尾声:牢固连接

常规：

  • 与团队中的其他增益职业配合，确保增益不会有断缺，防止使用 灵感纹章 时没有延长到关键性增益。
  • 注意使用粉碎技能用以触发 时间涌流 ，并略微提升输出。
  • 为了保证队友安逸的输出环境，有时你必须让 BOSS 背对大团方向，同时要将法池放在大团的一侧，而不是自己脚下。
  • 行动法池 与 回想法池 有一个持续 3秒的脉冲效果，3秒后才能为队友提供增益，如果你的队友比较分散，请及时提醒他们。
  • 开场 或 转阶段开场后，队友身上还未来得及覆盖增益时，不要着急使用 灵感纹章 。
  • 如果在对抗 BOSS 的致命攻击时，你将 圣盾 过早施放，可能需要使用 灵感纹章 来延长其生效时间，这样将有可能导致增益覆盖中断，但即使如此，也要比队友倒地再复活起来强很多。
  • 作为坦克拉 BOSS 时，尽可能使用技能来格挡伤害，而不是 闪避 。
  • 幻象跳跃 → 移形换位 附带 定身 ，必要时可以用来控制小怪。
  • 凡尘帷幔 → 堕入虚无 允许将 5个小怪聚集，方便集中清理。
  • 单手剑 自动攻击第三段 心灵钻刺 附带增益移除效果，你可以保留分身，让它们使用普攻来加快增益移除速度。
  • 如果队友都能合理的规避伤害，则使用 幻光纹章 作为治疗技能，以加快幻影的生成速度，并提高输出。该技能通常会搭配输出装备使用。
  • 谦逊纹章 通常用于破 BOSS 蔑视使用， 重力陷阱 用于聚怪晕怪（防止那些会踢人的小怪阻碍玩家输出），并能兼顾破蔑视。
  • 纹章类技能在读条结束后会获得 1秒的 失真 ，可以在关键时刻规避 BOSS 的技能。
