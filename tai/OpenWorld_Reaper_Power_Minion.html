. 夺魂者 . Reaper

直伤生存夺魂者

  Hyido
  泰协内容维护小组
  2020.10.21
  未经作者授权，禁止转载

概述

  • 野外生存夺魂者 是一套经典的野外配置，在萌新初入泰瑞亚时面临着的装备不达标情况下也能打出不俗伤害，使用这套方案可以单挑绝大多数的野外 BOSS。
  • 注意使用 夺魂之幕 时，由于你的游戏画面会变成绿油油的，加上周围如果小怪过多，可能会看不清怪的攻击前摇（比如地面显示的黄圈），请不要过于沉迷，别踩着圈输出~

特性 与 技能

  • 偏输出（怨恨摄魂）
  • 偏续航（鲜血死亡）
  • 战吼流

特性

  • 由于夺魂前两次的平衡性改动，使得夺魂在生命之力的获取上得到很大改善， 无尽生命 与 苦难纹章 的配合可以在野外保持全程的帷幕状态。
  • 如果帷幕条子被消耗过快，可以选择使用 无尽生命 。

技能

  • 夺魂召唤出来的宝宝不是全程让它们自由发挥的，请留意宝宝们的技能。
  • 召唤骨魔 召唤后， 死尸僵化 的效果可以让敌人获得短时间的 定身 ，在自身血量不安全的情况下，可以命令你的骨魔宝宝给敌人来上一个 定身 ，自己走远一点利用斧头或者帷幕的1技能 生命撕裂 进行攻击。
  • 召唤影魔 召唤后， 鬼影降临 的效果可以让敌人获得长时间 目盲 ， 冰冻 以及 虚弱 ，在面对敌人即将攻击前，事先让影魔宝宝给它安排一下 目盲 与 冰冻 ，在怪下一次攻击击中你时，你可以免去一次受伤的机会。除此之外， 鬼影降临 还可以恢复 10%的生命之力。
  • 血肉魔像 召唤后， 冲锋陷阵 的效果可以作为破蔑视的良好手段（332点蔑视值），且该技能目标数量为10个，附带的 击飞 与 击倒 在你面对较多的小怪时也能获得短时间良好的输出机会。
  • 如果你已经良好掌握该配置，也可以将 召唤骨魔 替换为 怨恨纹章 ，从而进一步增加直伤输出能力。

特性说明

  • 一般夺魂野外的伤害是足够的，喜欢生存性能好一点，皮厚一点的可以尝试一下。

特性

  • 恨意护符 、 唤灵腐化 的存在让你的宝宝们获得更强大的生存能力与输出能力，但是与之交换的是你自身输出能力的下降，如果希望有一群肉得不行的宝宝帮你打怪时候分散怪的仇恨，可以考虑使用。
  • 不洁殉难 的存在可以让你在退出F1帷幕型态时候解除3个症状并获得额外的生命之力，但是请注意如果此时周围有数名队友，且他们身上有过多的症状时，你一旦进入F1帷幕，会把队友的Debuff给吸过来。注意留意自身的Debuff数量，以免因此把生命之力白白耗光！

技能

特性说明

  • 战吼流的玩法年代比较久远，目前版本也还是可以尝试一下，但是上手难度比较中等，伤害还算过得去

特性

技能

  • 一般面对两个以上的怪时，如果怪的位置有点分散，可以自身稍微走到与怪一条直线上，利用大剑5技能 黑暗抓攫 进行聚怪，而后施放大剑4技能 夜幕降临 与3技能 死亡漩涡 ，一是为了给怪打上 目盲 以保证自身安全，二来可以给自身增加接下来的伤害输出。
  • 当多个怪与你距离较近时，记得在进入F1 夺魂之幕 前优先施放 "死亡将临！" 与 "弱者退散！" ，以获得更高的输出加成。
  • "严寒刺骨！" 可以留在你退出了 夺魂之幕 的爆发后使用，保证在 夜幕降临 后给怪来个群控，获得更长时间的输出环境。

装备 与 消耗品

  • 通用
  • 战吼

装备说明

  • 初入游戏阶段若未能获得全套的升华装备（粉装），可以在拍卖行买一套特异装备（橙装）用来过渡。
  • 如果野外生存能力欠佳，可以考虑把符文或者部分装备更换成更具生存性的。

武器（可选）：

  • 聚能器：依靠 刺骨恶寒 去除敌人的增益。
  • 法杖：伤害较低，但通用效果很均衡，远程磨怪。
  • 选用 超级嗜血法印 的原因是，很多新手不会在野外特地吃食物与通用效果，而且自身装备不达到毕业标准， 超级嗜血法印 的效果在你击杀满25只怪时直接给你250威力加成，何乐而不为呢~（注意击杀时必须是这把打了 超级嗜血法印 的武器击杀，若切换到其他武器组，将无法获得buff加成）
  • 其他法印位置如果多出来，可以根据自身经济能力选用 超级武勇法印 或者其他法印。

符文：

  • 超级吸血符文 ：牺牲一部分伤害能力，换取更强的续航。
  • 超级狼群符文 ：良好的属性加成。
  • 超级霍布雷克符文 ：便宜，你懂的。
  • 超级学者符文 ：毕业级符文，价格偏高，视自身情况选用。

配置说明

技能使用

伤害技能：

  • 夺魂之幕 可以提供巨额的输出，在拥有充足的生命之力时，你就应该进入帷幕状态，以获得 急速 和 高额的AOE伤害。（ 夺魂灭屠 ）

    • 不要打断 夺魂之幕 的自动攻击链： 生命撕裂 → 生命劈裂 → 生命收割

  • 夜幕降临 和 灵魂漩涡 是你伤害最高的技能，在近战的战斗中，使用它们收割敌人。

  • 一旦你的爆发型技能全部用尽，就应该使用 掘墓剑斩 ，并尽可能多的使用它。在敌人生命值低于 50%时，该技能将没有冷却时间。
  • 当 夜幕降临 、 掘墓剑斩 、 夺魂之幕 都无法使用时，切换到巨剑并使用所有技能。
  • 斧/战号 的持续伤害很低，但可以施放 恐怖之爪 和 迅捷蝗群 两个高伤害技能，你可以在施放这些技能后就切换到巨剑，或是在进入 夺魂之幕 前施放。

防御技能：

  • 在面对一大波敌人时，记得让你的仆从吸引火力。通过 闪避 和 拉扯 尽可能避免受到敌人的攻击，因为如果在 夺魂之幕 时，受到攻击会消耗生命之力，切回本体时，受伤也会影响 超级学者符文 的被动。
  • 夜幕降临 、 死亡冲刺 和 鬼影降临 都可用于为敌人施加 目盲 。
  • 如果短时间内受到大量伤害，可以切换到 夺魂之幕 ，然后使用 惊惧魔甲 降低伤害并获得 稳固 。
  • 如果生存依然有压力，可以考虑点出特性 堕落狂热 。

说明

续航能力：

  • 可以通过 召唤血魔 或 “鲜血”特性分支 获得持续的小额治疗。
  • 大多数情况下，你都可以依靠 唤灵腐化 将自身的症状转移出去，如果需要更多清症能力，可以点出 幕化移除 ，或 佩戴副手匕首使用 致命虫群 转移症状。
  • 使用主手匕首时，可以不断使用 生命虹吸 治疗自己。
  • 如果不小心被打入倒地状态，尝试通过 生命榨取 重新站起来。

控制技能（破蔑视）：

  • 黑暗抓攫
  • 末日哀嚎
  • 冲锋陷阵
  • 惊惧释放
  • 处决镰刀
  • 鬼影降临
  • 死尸僵化
